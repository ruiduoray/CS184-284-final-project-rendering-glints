Final Project: Rendering Glints for Path Tracer

Proposal

Summary

Our Project aims to add texture supports and mipmapping to our existing path tracer projects. Then, we will add glint effects by using high resolution normal maps.

Team Member

Weili Liu, Frank Cai, Ray Gong, Xuantong Liu

Problem Description

For complex specular surfaces under sharp point lighting, it is impractical to use Monte Carlo pixel sampling to render them, because energy is concentrated on tiny highlights. The problem is important because we would like to render more realistic images. Therefore, we plan to use a method which calculates on the true distribution of normals on a surface patch to solve this problem.

Goals and Deliverables

We would like to be able to render glints on complex specular surfaces just like the real world. We should be able to deliver glinty appearance in still images.

Schedule

  • Week 2: Implementing major functionalities
